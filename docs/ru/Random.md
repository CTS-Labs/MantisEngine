# Документация модуля Random

Этот документ предоставляет всесторонний обзор модуля `random`, который включает различные функции для генерации случайных значений и манипуляции таблицами.

## Обзор

Модуль `random` включает следующие функции:

- `randomElement(tbl)`
- `randomElements(tbl, count)`
- `randomInteger(min, max)`
- `randomFloat(min, max)`
- `shuffle(tbl)`

Каждая функция предназначена для обработки случайных выборок и генерации чисел с определенными ограничениями.

## Функции

### 1. `random.randomElement(tbl)`

Возвращает случайный элемент из предоставленной таблицы.

#### Параметры
- `tbl`: Таблица (массив), из которой выбирается случайный элемент.

#### Возвращает
- Случайно выбранный элемент из `tbl`.

#### Ошибки
- Генерирует ошибку, если таблица пуста.

#### Пример
```lua
local fruits = {"apple", "banana", "cherry"}
local randomFruit = random.randomElement(fruits)
```

### 2. `random.randomElements(tbl, count)`

Возвращает заданное количество уникальных случайных элементов из предоставленной таблицы.

#### Параметры
- `tbl`: Таблица (массив), из которой выбираются элементы.
- `count`: Количество случайных элементов для возврата.

#### Возвращает
- Таблицу, содержащую `count` уникальных случайных элементов из `tbl`.

#### Ошибки
- Генерирует ошибку, если:
  - Таблица пуста.
  - `count` меньше 1 или больше размера таблицы.

#### Пример
```lua
local numbers = {1, 2, 3, 4, 5}
local randomNumbers = random.randomElements(numbers, 3)
```

### 3. `random.randomInteger(min, max)`

Генерирует случайное целое число между заданными минимальным и максимальным значениями.

#### Параметры
- `min`: Минимальное целое значение (включительно).
- `max`: Максимальное целое значение (включительно).

#### Возвращает
- Случайное целое число между `min` и `max`.

#### Ошибки
- Генерирует ошибку, если `min` больше `max`.

#### Пример
```lua
local randomInt = random.randomInteger(1, 10)
```

### 4. `random.randomFloat(min, max)`

Генерирует случайное число с плавающей точкой между заданными минимальным и максимальным значениями.

#### Параметры
- `min`: Минимальное значение с плавающей точкой.
- `max`: Максимальное значение с плавающей точкой.

#### Возвращает
- Случайное число с плавающей точкой между `min` и `max`.

#### Ошибки
- Генерирует ошибку, если `min` больше `max`.

#### Пример
```lua
local randomFloat = random.randomFloat(0.5, 1.5)
```

### 5. `random.shuffle(tbl)`

Случайным образом перемешивает элементы предоставленной таблицы на месте.

#### Параметры
- `tbl`: Таблица (массив), которую нужно перемешать.

#### Возвращает
- Перемешанную таблицу.

#### Пример
```lua
local cards = {"Ace", "King", "Queen", "Jack"}
random.shuffle(cards)
```
